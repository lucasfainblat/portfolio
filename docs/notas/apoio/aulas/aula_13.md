# Aula 13: Abrindo uma issue
🔗 [Vídeo](https://www.youtube.com/watch?v=bB2RteacjxQ)

## Notas
- Uma `issue` é um tópico de discussão referente a uma tarefa pendente, um erro identificado etc.
- Abrir uma `issue` corresponde à formalização verbal de um ponto de melhoria, comentário, erro, lista de tarefas etc.
- Criei duas issues no repositório do portfólio e aproveitei pra criar um blog pra completar a atividade da próxima aula.
